Documentation API

Le projet Ilyox'SSH utilise plusieurs bibliothèques et API Python afin d’assurer la connexion sécurisée, l’affichage en terminal et la génération d’exécutables Windows.

1. Paramiko (SSH)

Paramiko est une bibliothèque Python qui implémente le protocole SSH2. Elle permet :

  • Connexion à des serveurs distants via ssh.connect()
  • Exécution de commandes distantes via ssh.exec_command()
  • Gestion des clés privées et authentification par mot de passe

Exemple :

import paramiko

ssh = paramiko.SSHClient()
ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
ssh.connect("example.com", username="root", password="secret")

stdin, stdout, stderr = ssh.exec_command("ls -la")
print(stdout.read().decode())
ssh.close()

2. Colorama (Couleurs terminal)

Colorama permet d’afficher du texte coloré dans le terminal, compatible Windows et Linux.

Exemple :

from colorama import Fore, Style

print(Fore.GREEN + "Succès !" + Style.RESET_ALL)
print(Fore.RED + "Erreur !" + Style.RESET_ALL)

3. Pyfiglet (ASCII Art)

Pyfiglet génère du texte en ASCII Art (grands caractères stylisés).

Exemple :

import pyfiglet

ascii_banner = pyfiglet.figlet_format("Ilyox'SSH")
print(ascii_banner)

4. PyInstaller (Executable .exe)

PyInstaller transforme un script Python en exécutable Windows (.exe). Cela permet d’utiliser Ilyox'SSH sans installer Python.

Commande :

pyinstaller --onefile --console ilyox_ssh.py

Résumé

  • Paramiko → SSH et exécution de commandes
  • Colorama → Couleurs et mise en forme terminal
  • Pyfiglet → ASCII Banner personnalisé
  • PyInstaller → Génération d’exécutables
