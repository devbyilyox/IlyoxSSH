Ilyox'SSH - Documentation

📖 Introduction

Bienvenue dans la documentation de Ilyox'SSH. Ce terminal SSH personnalisé vous permet de vous connecter à vos serveurs Linux depuis Windows avec une interface simple, moderne et un banner ASCII unique.

⚙️ Installation

  1. Téléchargez l’exécutable IlyoxSSH.exe.
  2. Lancez l’application en double-cliquant dessus.
  3. Aucune installation supplémentaire n’est requise.

🔑 Connexion SSH

  • IP du serveur (ex: 123.45.67.89)
  • Port (par défaut 22)
  • Utilisateur (ex: root)
  • Mot de passe ou clé privée

💻 Utilisation

root@123.45.67.89$ ls -la

requirements.txt

Utilisez exit pour fermer la session.

📦 Exemple : installation d’un bot Discord

# Mettre à jour les paquets
sudo apt update && sudo apt upgrade -y

# Installer Python et Git
sudo apt install -y python3 python3-venv python3-pip git

# Cloner votre bot
git clone https://github.com/utilisateur/monbot.git

# Lancer le bot
python3 bot.py
